2048cpp
=======

2048 game, my first C++ Application

compile: g++ game.cpp main.cpp -lcurses -o 2048

gcc 4.6.3 and possibly older:
g++ -std=c++0x game.cpp main.cpp -o 2048

Made for OSX 10.9

I hope somebody find it useful or fun.
